Python, qu'est-ce que c'est ?


Python est un langage de programmation plutôt simple et dit "orienté objet" c'est à dire qu'il s'articule autour d'objets et de données, plutôt que d'actions et de logique. De plus, il peut être utilisé dans de nombreux contextes et s'adapter à tout type d'utilisation grâce à des bibliothèques spécialisées. Il est également apprécié par certains pédagogues qui y trouvent un langage où la syntaxe, clairement séparée des mécanismes de bas niveau, permet une initiation aisée aux concepts de base de la programmation.   # Exemple de la suite de Fibonacci (récursive) !
  def fibonacci_recursif(n):
     if n < 2:
         return n 
     return fibonacci_recursif(n-1) + fibonacci_recursif(n-2)
  
  resultat = fibonacci_recursif(10)
  print(f{"Le résultat est : {resultat}"}) 
  # Renvoie le 10e élément de la suite de Fibonacci : 55

Les Avantages de Python

Simple d'utilisation

Tout d'abord, Python est un langage simple, puissant et facile à apprendre. Sa syntaxe et son grand nombre de bibliothèques permettent d'avoir un code clair et de faire énormément de choses !

Intelligence Artificielle

De plus, Python est très utile pour les IA et propose une approche simple pour les Moyens / Débutants, notamment encore une fois grâce à son grand nombre de bibliothèques.

Sciences

Enfin, Python est un langage très utile en science (des données, physique, mathématiques). Il est le plus répandu dans ce domaine et il est parfaitement adapté pour l'analyse de données en sciences de la vie.
