Les bases de Python


Pour commencer, voici les opérations basiques :


            
L'addition

        
  var = 0
  var = var + 1 # ou var += 1
  print(var) # Affiche 1

        
            
La soustraction

        
  var = 0
  var = var - 1 # ou var -= 1
  print(var) # Affiche -1

        
            
La multiplication

        
  var = 2
  var = var * 2 # ou var *= 2
  print(var) # Affiche 4

        
            
La division

        
  var = 16
  var = var / 2 # ou var /= 2
  print(var) # Affiche 8.0

Ensuite, voici des opérations un peu plus complexe :

            
Les puissances

        
  var = 4
  var = var**2 # ou var **= 2
  print(var) # Affiche 16

        
            
Le Modulo (Reste de la division)

        
  var = 11
  var = var % 10 # ou var %= 10
  print(var) # Affiche 1

        
                
La division (entière)

            
      var = 7
      var = var // 2 # ou var //= 2
      print(var) # Affiche 3
